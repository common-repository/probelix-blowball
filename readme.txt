Easily integrate Blowball in Wordpress.

== Description ==

Blowball is a middleware solution for businesses with multiple functions for e-commerce, product information management (PIM), marketing and CRM.
With this plugin you'll connect Wordpress with Blowball and include different functions on your website.

Currently the following functions are supported:

* Connection with Contact Form 7 plugin. Contacts submitting a form via Contact Form 7 can be synchronized with Blowball.

PS: For this plugin you need your own [Blowball instance](https://blowball.io) to connect to.
